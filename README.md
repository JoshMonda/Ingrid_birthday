- Sparkle effects

## 🚀 How to Deploy

1. **Simple Deployment**: Just upload all 3 files to any web server:
   - `index.html`
   - `styles.css` 
   - `script.js`

2. **Local Testing**: Open `index.html` in any modern web browser

3. **No Dependencies**: Everything is self-contained - no external libraries needed!
